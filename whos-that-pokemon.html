Who's That Pokémon?

Project information

  • Project Type: DigitalCrafts Atlanta solo project
  • Project Date: December 2021
  • Project URL: placeholder
  • GitHub Repo: Who's That Pokémon?
  • Technologies Used: HTML, CSS, React, Redux

Re-live your childhood memories by using this pokédex!

"Who's That Pokémon?" lets users search for their favorite pokémon and build their own poké team. React is the backbone of this app, leveraging multiple custom-built components to create the pokédex in the UI. React state changes in the Search and Teams components and is updated using Redux. All information is sourced from the PokéAPI.

My biggest hurdle was setting up Redux. The relationships between actions, reducers, took some time to fully understand. Not to mention effectively keeping track of all the Redux files. I spent a considerable amount of time outside of class doing research.

This project also really stretched my CSS knowledge. From border-radius to SVGs to absolute/relative positioning – my eyes were opened into how powerful and complex CSS can be (CSS specificity, anyone?).
