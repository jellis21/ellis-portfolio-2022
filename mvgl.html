MyVideoGameList

Project information

  • Project Type: DigitalCrafts Atlanta solo capstone project
  • Project Date: January 2022
  • Project URL: my-video-game-list.herokuapp.com
  • GitHub Repo: MVGL
  • Technologies Used: HTML, CSS, React, Node.js with Express, JavaScript, PostgreSQL with Sequelize, Bootsrap React

Create a curated list of your favorite video games!

MyVideoGameList is a social cataloging application allowing users to search for video games. Search results come from the IGDB API and are mapped over Bootstrap cards (the search results container, including functionality to move through the results, is custom made). The app is fully responsive with different landing page layouts for mobile, tablet, and computer viewport widths.

Once the user is logged in they can add games to their list, assign each game a score, and perform all CRUD functionality. Additionally, logged-in users can view the lists other users create. Security is of the utmost importance: passwords are stored as a hash to a PostgreSQL database using the bcrypt npm package while login functionality leverages the JSON Web Token standard.
